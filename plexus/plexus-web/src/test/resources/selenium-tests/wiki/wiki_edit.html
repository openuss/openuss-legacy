type xpath=//textarea[contains(@id, 'editWikiForm')] <h1>Wiki</h1>
<p>edited</p>

<p>Ein <strong>Wiki</strong> (<a href="http://de.wikipedia.org/wiki/Hawaiische_Sprache">Hawaiisch</a> f&uuml;r &bdquo;schnell&ldquo;<sup><a href="http://de.wikipedia.org/wiki/Wiki#_note-0">[1]</a></sup>), seltener auch <em>WikiWiki</em> und <em>WikiWeb</em> genannt, ist eine Sammlung von <a href="http://de.wikipedia.org/wiki/Webseite">Webseiten</a>, die von den Benutzern nicht nur gelesen, sondern auch direkt <a href="http://de.wikipedia.org/wiki/Online">online</a> ge&auml;ndert werden k&ouml;nnen. Wikis erm&ouml;glichen es verschiedenen Autoren, gemeinschaftlich an Texten zu arbeiten.</p>

<p>Wesentlich bei der meisten <a href="http://de.wikipedia.org/wiki/Wiki-Software">Wiki-Software</a>, darunter das in der <a href="http://de.wikipedia.org/wiki/Wikipedia">Wikipedia</a> selbst eingesetzte <a href="http://de.wikipedia.org/wiki/MediaWiki">MediaWiki</a>, ist die <a href="http://de.wikipedia.org/wiki/Versionsverwaltung">Versionsverwaltung</a>, die es den Benutzern im Fall von Fehlern oder <a href="http://de.wikipedia.org/wiki/Vandalismus">Vandalismus</a> erlaubt, eine fr&uuml;here Version einer Seite wiederherzustellen. Wie bei <a href="http://de.wikipedia.org/wiki/Hypertext">Hypertexten</a> &uuml;blich, sind die einzelnen Seiten eines Wikis durch Querverweise (<a href="http://de.wikipedia.org/wiki/Hyperlink">Hyperlinks</a>) miteinander verbunden; zur Vernetzung von verschiedenen Wikis dient das Konzept der InterWiki-Verweise.</p>

<p>Wikis geh&ouml;ren zu den <a href="http://de.wikipedia.org/wiki/Content-Management-System">Content-Management-Systemen</a>, setzen aber auf die Philosophie des offenen Zugriffs, im Unterschied zu teils genau geregelten <a href="http://de.wikipedia.org/wiki/Arbeitsablauf">Arbeitsabl&auml;ufen</a> (workflow) von Redaktionssystemen.<sup><a href="http://de.wikipedia.org/wiki/Wiki#_note-1">[2]</a></sup> Mit &bdquo;offenem Zugriff&ldquo; ist aber nicht gemeint, dass zwangsl&auml;ufig jedes Wiki f&uuml;r alle lesbar oder schreibbar sein m&uuml;sste, es kann sehr wohl Berechtigungen oder Beschr&auml;nkungen f&uuml;r bestimmte Benutzergruppen geben (vgl. <a href="http://de.wikipedia.org/wiki/Wiki#Typen">Typen</a>).</p>

<p>Die &Auml;nderbarkeit der Seiten durch jedermann setzt eine urspr&uuml;ngliche <a href="http://de.wikipedia.org/wiki/Idee">Idee</a> des <a href="http://de.wikipedia.org/wiki/World_Wide_Web">World Wide Web</a> erstmals konsequent um und erf&uuml;llt weiterhin eine wesentliche Anforderung an <a href="http://de.wikipedia.org/wiki/Soziale_Software">soziale Software</a>.</p>
