CaseStudy Help

Working with the Case Study is much like working with the Text Study. You should walk here when you feel familiar with the learning content and want to test your knowledge by finding solutions for complex cases setup in the Case Study View. Selecting any non-folder element from the Structure Tree will display the corresponding HTML document in the Content Panel.


The Interaction Panel contains Default Interaction Buttons: the Default Navigation Buttons and the Split Screen Buttons which allow to navigate back and forth through the Structure Tree and view two Text Elements at the same time.
