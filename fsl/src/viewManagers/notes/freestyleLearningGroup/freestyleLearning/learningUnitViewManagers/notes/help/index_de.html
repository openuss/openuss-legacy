Notes Manager Hilfe

Beim Notiz-Manager handelt es sich um eine Implementierung einer sekundären Perspektive. Sofern Ihre aktuelle Learning Unit diese View unterstützt, finden Sie eine zusätzliche Schaltfläche im rechten Bereich der Interaktionsleiste bei jeder anderen View: den Context-Dependent Activation Button für den Notiz-Manager. In dieser Schaltfläche wird ein leeres oder ein beschriebenes Blatt Papier symbolisch dargestellt. Wenn ein leeres Blatt zu sehen ist, gibt es noch keine Notizen zu einem Element. Ein beschriebenes Blatt deutet an, dass zu dem gerade betrachteten Element bereits eine Notiz existiert.



Sie können direkt zu der Notiz wechseln, indem Sie den Context-Dependent Activation Button drücken. Um eine Notiz editieren zu können, müssen Sie sich im Editier-Modus befinden. Die Arbeit mit den Notiz-Elementen funktioniert ähnlich wie mit den Text Study-Elementen. Diese können ebenso Hypertext-Dokumente mit formatiertem Text, Bildern, Tabellen und Links ins Internet oder zu anderen Freestyle Learning Views enthalten.
