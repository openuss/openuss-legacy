Glossary Help

The Glossary represents a reference for terms used in hypertext documents of different Learning Unit Views, like the Text Study. In fact it is quite similar to the Text Study View in terms of usage. Once you are familiar with the Text Study, you will also feel comfortable with the Glossary. The main difference is that the Glossary contains no chapter like structure, in most cases it is just a collection of terms and definitions.


The Interaction Panel contains Default Interaction Buttons: the Default Navigation Buttons and the Split Screen Buttons which allow to navigate back and forth through the Structure Tree and view two Text Elements at the same time.
