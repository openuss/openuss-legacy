Media Pool Help

The Media Pool is a collection of files containing additional information and presentation for a specific Learning Unit. It is a Secondary View, so you are not likely to walk through it in a linear manner. In most cases you will be directed to Media Pool Elements by following links from any hypertext document of a Primary View, the Text Study for example. But you are also free to quarry through the Media Pool and start your personal exploration!


The following types of MultiMedia Elements are available

  • Picture Files
  • Audio Files
  • Video Files
  • Acrobat PDF Files
  • Power Point Presentation Files
  • Executable Files

Depending on the type of element you selected from the Structure Tree, there will appear some of the Interaction Buttons below.


Start playing a selected audio, video or executable file (Acrobat, PowerPoint presentation)


Pause while playing audio or video files


Stop and reset playback of audio or video file
