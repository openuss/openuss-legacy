Media Pool Hilfe

Der Media Pool ist eine Ansammlung von Dateien mit zusätzlichen Informationen zu einer spezifischen Learning Unit. In den meisten Fällen werden Sie direkt durch einen Link im entsprechenden Hypertext-Dokument des primären Views zu dem entsprechenden Multimedia-Element im Media Pool geleitet. Natürlich können Sie auch direkt den Media Pool aufrufen und Ihre individuelle Entdeckungsreise starten.


Die folgenden Multimediaformate werden unterstützt:

  • Bildformate
  • Audiodateien
  • Videodateien
  • Acrobat PDF-Dateien
  • PowerPoint Präsentationsdateien
  • Executable Files

Abhängig von dem gewählten Format aus dem Structure Tree, werden die entsprechenden Interaction Butttons im Interaction Panel angezeigt.
