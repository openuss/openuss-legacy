Learning by Doing Help

When you worked your way through the Text Study, Slide Show or any other View to gain basic and detailed knowledge about the Learning Unit Content, you can also work interactivly with experimental setups like executable files or other Java applications which may simulate or illustrate the presented learning content. You find these files selectable from the Structure Tree of the Learning By Doing View.


If an Application File Element is selected, you can start the appropriate application over the "Start"-Button in the Interaction Panel.
