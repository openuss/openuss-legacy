Welcome to

This document will guide you through your first steps using the Freestyle Learning Home software.
To get started, you have to perform the following steps:

  1. Log in with your user name and password. After installing, at first starup the login will perform automatically.
  2. Select a Learning Unit
  3. Press a Primary or Secondary Activation Button to select a Learning Unit View
  4. Select an element from the Structure Tree to view the specific content

Once you selected a Learning Unit View, you can navigate through your selected elements with the Main Navigation Buttons places in the toolbar.
