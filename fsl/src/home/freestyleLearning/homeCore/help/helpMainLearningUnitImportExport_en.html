Import and Export of Learning Units

Freestyle Learning organizes the learning content into Learning Units. Learning Units can be imported / exported to exchange with other users.

Learning Unit Import

Learning Units can be uploaded and downloaded, e. g. by using the Online-Platform OpenUSS. Therefore Freestyle Learning uses .fsl-files. To import these files you have to select "File / Import" from the menu. In the upcoming dialog you have to search for .the fsl-file to import. After a successful import process you can activate the Learning Unit from the Learning Unit Structure Tree.

Hint: Keep in mind that a Learning Unit Import of a Learning Unit with the ID of a currently installed Learning Unit is not possible. In this case you have to delete the Learning Unit with the same ID.

Learning Unit Export

If author-role is enabled you can also export Learning Units to provide the Learning Unit contents for other Freestyle Learning users. Therefore you have to activate the unit to export by using the Learning Unit Structure Tree Dialog. Then you have to select "File / Export". Alternatively you can export a Learning Unit by a right click on the Learning Unit name in the Learning Unit Structure Tree which invokes a context menu. Choose "Export" to start the export process. In the upcoming dialog you can set a name and the target directory for the .fsl-file. Click on "Save".

Hint: Please keep in mind that the Learning Unit to export can include links on elements from Learning Units, which might be not installed on the Freestyle Learning platforms from other users. Such links can not be dissolved.
