Import und Export von Learning Units

Importieren einer Learning Unit

Lerninhalte können in Form von Learning Units für die Freestyle Learning-Plattform zum Download angeboten werden, z. B. über eine entsprechende Veranstaltung, die auf der Online-Plattform OpenUSS eingerichtet ist. Der Download einer Unit erfolgt in einer einzigen Datei mit der Endung .fsl (zip-Format). Zum Importieren einer Unit aus einer .fsl-Datei wählen Sie den Menüpunkt "Datei / Importieren". Im darauf erscheinenden Dialog werden Sie zur Auswahl einer .fsl-Datei aufgefordert. Bestätigen Sie Ihre Auswahl, um den Import-Prozess zu starten. Nach erfolgreichem Import können Sie die neue Unit über den Learning Unit-Auswahldialog aktivieren.

Hinweis: Beachten Sie, dass ein Import einer Unit mit bereits auf Ihrer Plattform verwendeter ID derzeitig nur möglich ist, wenn zuvor die existierende Unit mit der gleichen ID gelöscht wurde.

Exportieren einer Learning Unit

Sofern Sie in der Autorrolle angemeldet sind, können Sie Ihre Units über die Exportfunktion leicht anderen Nutzern von Freestyle Learning zur Verfügung stellen. Aktivieren Sie dazu die zu exportierende Unit über den Learning Unit-Auswahldialog und wählen Sie im Menü "Datei / Exportieren". Alternativ können Sie im Learning Unit-Auswahldialog eine Unit rechtsklicken und "Exportieren" wählen. Im darauf erscheinenden Dialog können Sie einen Namen für die .fsl-Datei festlegen und den Exportvorgang starten. Diese Datei können Sie dann z. B. für andere Nutzer von Freestyle Learning zum Download anbieten, so dass diese sie wiederum über die Importfunktion bei sich nutzen können.

Hinweis: Beachten Sie, dass eine exportiere Unit keine Links auf Elemente von Units enthält, die bei den importierenden Installationen ggf. nicht vorhanden sind. Solche Links können dann nach dem Importieren nicht aufgelöst werden.
