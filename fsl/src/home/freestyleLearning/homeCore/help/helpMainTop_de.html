Dieses Hilfe wird Sie bei Ihren ersten Schritten in der Freestyle Learning Software unterstützen. Zum Einstieg gehen Sie in folgenden Schritten vor:

  1. Melden Sie sich mit Ihrem Benutzernamen und Passwort an. Dies erfolgt nach der Installation beim Programmstart automatisch.
  2. Wählen Sie eine Learning Unit aus dem
  3. Klicken Sie einen Primary oder Secondary Activation Button um eine Learning Unit View auszuwählen
  4. Wählen Sie ein Element aus dem Structure Tree zum Betrachten des dazugehörigen Inhaltes

Wenn Sie eine Learning Unit View gewählt haben, können Sie sich mit den Main Navigation Buttons aus der Toolbar durch die gewählten Elemente navigieren.
